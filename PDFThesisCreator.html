PDF Thesis Creator

Get automatically structured PDF files by taking advantage of LaTeX

Description

PDF Thesis Creator is an Android App that allows to manage .tex files and plain text notes by choosing, through a simple UI, the items to include in an automatically generated LaTeX master file with a predefined style and obtain a structured PDF file directly on your Android device.

Main Activity

From the main activity, as shown in fig.1, you can start the other activities through the options menu.

User Defined Settings

By clicking on Settings you can modify the default Filename and its document properties (Title, Author, Subject and Keywords).

Import Notes

By clicking on Import notes you can import .txt and/or .tex files from the directory:

/sdcard/Txt2TexINPUT/

Create PDF File

From Create PDF File you get the list of all items, in which, as shown in fig.2, you can select them and through the button Create PDF File With Selected Items create the output directory:

/sdcard/Txt2TexOUTPUT/FilenameYYYYMMdd_hhmmss/

with automatically generated LaTeX master and pdf files (look at Outputs for more details).

Internal text editor

From the main activity you can also click one of the items to edit them with an internal text editor, and by clicking on New txt and New tex you can add new .txt and .tex notes.

Delete

By clicking on Delete you can select the notes to delete from the application database.

Inputs

Imported Notes

Imported notes should be in UTF-8 encoding.

Filenames of .txt notes

The .txt notes are included in the master file according to their filenames, for example 01-A_sample_of_.txt_note.txt:

\chapter{A sample of .txt note}
\VerbatimInput{\detokenize{./FILES/01-A_sample_of_.txt_note.txt}}

Filenames of .tex files

Even the .tex files are included in the master file according to their filenames, for example 00-A_sample_of_.tex_note.tex:

\chapter{A sample of .tex note}
\input{\detokenize{./FILES/00-A_sample_of_.tex_note.tex}}
but if you want to put the title of the chapter within the .tex note, you have to put the string "Chapter" in the filename of the .tex note, for example Chapter1.tex:

\input{./FILES/Chapter1.tex}

Title with empty spaces

To obtain empty spaces in the title of a chapter in the LaTeX master file you have to put an underscore “_” in the filename of the note (e.g. A_sample_of_.txt_note.txt).

Alphabetical or user defined order of notes

By default the notes (.txt and/or .tex) are inserted in the master file according to alphabetical order, but you can add numeric indexes, followed by a minus sign “-”, at the beginning of the filename to get a different order (e.g. 01-A_sample_of_.txt_note.txt, 00-ChapterA.tex).

Inserting images in .tex notes

You can insert images in .tex notes from the directory:

/sdcard/Txt2TexINPUT/IMAGES/.

Output

The output directory /sdcard/Txt2TexOUTPUT/FilenameYYYYMMdd_hhmmss/ (within the DeviceStorage) contains:

  • a PDF file: notes.pdf (as shown in fig.3);
  • a LaTeX master file: notes.tex;
  • a folder with selected .txt and/or .tex notes: FILES/;
  • a folder with images: IMAGES/;
  • a predefined style file: defaultstyle.sty.

The default style file

In the current version you get a predefined style file defaultstyle.sty in the output directory /sdcard/Txt2TexOUTPUT/FilenameYYYYMMdd_hhmmss/ that can be modified with an external editor.
