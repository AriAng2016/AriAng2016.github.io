LaTeX Master File Creator

Automatic generation of compilable LaTeX master files in Android

Description

LaTeX Master File Creator is an Android App that allows to automatically insert .txt and/or .tex files in a compilable LaTeX master file with sorted sections as:

\documentclass{...}
   ...
\begin{document}
\section{Topic A}
   ...
\section{Topic B}
   ...
\section{Topic C}
   ...
\section{Topic D}
   ...
\end{document}

according to their filenames as shown in fig.1.

From NotesList activity you can start the other activities through the options menu:

  • 1.Settings
  • 2.Import notes
  • 3.Get topics
  • 4.New txt
  • 5.New tex
  • 6.Delete

User Defined Settings

By clicking on 1.Settings you can modify the default Filename of LaTeX master file and its document properties (Title, Author, Subject and Keywords) as shown in fig.2.

Import Notes

By clicking on 2.Import notes you can choose the files .txt and/or .tex in the input directory: /sdcard/Txt2TexINPUT/ and import them through the button Import Selected Notes as shown in fig.3.

Get Topics

From 3.Get Topics you get the list of all topics, in which, as shown in fig.4, you can select them and through the button Get compilable LaTeX master file with selected topics create the output directory /sdcard/Txt2TexOUTPUT/YYYYMMdd_hhmmss/ with an automatically generated LaTeX master file (look at Outputs for more details).

By using the button Get notes with selected topics you get the list of the files with the chosen topics.

Internal text editor

Still from the main activity NotesList you can click one of the items to edit them with an internal text editor, and by clicking on 4.New txt and 5.New tex you can add new .txt and .tex notes.

Delete

By clicking on 6.Delete you can select the notes to delete from the application database.

Input

Imported Notes

Imported notes should be in UTF-8 encoding.

Title with empty spaces

To obtain empty spaces in the title of a section in the LaTeX master file you have to put an underscore “_” in the filename of the note (e.g. Example_of_a_Section_Title.txt).

Alphabetical or user defined order of notes

By default the notes (.txt and/or .tex) are inserted in a compilable Latex master file according to alphabetical order, but you can add numeric indexes, followed by a minus sign “-”, at the beginning of the filename to get a different order (e.g. 01-Topic_Z.txt) as shown in fig.5.

Inserting images in .tex notes

You can insert images in .tex notes from the directory /sdcard/Txt2TexINPUT/IMAGES/.

Output

The output directory /sdcard/Txt2TexOUTPUT/YYYYMMdd_hhmmss/ (within the DeviceStorage) contains:

  • a LaTeX master file: notes.tex;
  • a folder with selected notes .txt and/or .tex: FILES/;
  • a folder with images: IMAGES/;
  • a predefined style file: defaultstyle.sty.

The default style file

In the current version you get a predefined style file defaultstyle.sty in the output directory /sdcard/Txt2TexOUTPUT/YYYYMMdd_hhmmss/ that can be modified with an external editor.

PS. You have to use an external LaTeX App to compile (with pdflatex) your LaTeX master file and generate a .pdf file as shown in fig.6
